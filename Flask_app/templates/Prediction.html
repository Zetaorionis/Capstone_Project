Let's Make a Prediction





What is this model predicting?

Our model attempts to predict whether a wine has a rating of 90 points or more given several features about the wine (price, country, variety and vintage).

Visit Wine Enthusiast for more information on Wine Rating Categories

What data is the model using?

When you enter an integer in the form above, the predicitve model will be given a wine with that corresponding row number(index) from the test dataset we used to test the accuracy of our model (i.e. if you enter 3, the model will make a prediction on the wine in the test dataset that has a row index of 3).

What will I see when I submit the form?

When you submit the form above, the results page will display the wine you sent to the model to predict, including the features used to make the prediction along with some additional metadata from the original/cleaned dataset used in our data analysis.

The page will also show the actual Rating(points), what the model predicted (Is the wine 90 points or above?), and a statement about the model's accuracy for that wine.
